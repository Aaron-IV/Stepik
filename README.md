# Шпаргалка по форматным спецификаторам в Go

Эта шпаргалка поможет тебе быстро ориентироваться в форматных спецификаторах в языке Go для использования с функцией `fmt.Printf` и другими методами.

## 1. Строки

- **`%s`** — форматирует строку (обычный текст)

  ```go
  fmt.Printf("%s", "Привет, мир!")  // Привет, мир!


>   %q — форматирует строку с кавычками (fmt.Printf("%q", "Привет")  // "Привет")

2. Целые числа (int)

>   %d — для целых чисел в десятичной системе (fmt.Printf("%d", 123)  // 123)
>   %b — для вывода числа в двоичной системе (fmt.Printf("%b", 12)  // 1100)
>   %o — для вывода числа в восьмеричной системе (fmt.Printf("%o", 10)  // 12)
>   %x — для вывода числа в шестнадцатеричной системе (малые буквы) (fmt.Printf("%x", 255)  // ff)
>   %X — для вывода числа в шестнадцатеричной системе (большие буквы) (fmt.Printf("%X", 255)  // FF)

3. Вещественные числа (float)

>   %f — для вещественных чисел, выводит 6 знаков после запятой по умолчанию (fmt.Printf("%f", 3.14159)  // 3.141590)
>   %.2f — для вывода вещественного числа с двумя знаками после запятой (fmt.Printf("%.2f", 3.14159)  // 3.14)
>   %g — для более компактного представления вещественного числа (например, если оно очень маленькое или большое) (fmt.Printf("%g", 0.00001234)  // 1.234e-05)
>   %e — для вывода в научной записи (с использованием экспоненты) (fmt.Printf("%e", 123456.78)  // 1.234568e+05)
>   %E — то же, что и %e, но в верхнем регистре (fmt.Printf("%E", 123456.78)  // 1.234568E+05)

4. Числа с фиксированной точностью

>   %5d — для целых чисел, где ширина поля 5 знаков (fmt.Printf("%5d", 42)  // "   42")
>   %-5d — для целых чисел, выравнивание по левому краю в поле шириной 5 (fmt.Printf("%-5d", 42)  // "42   ")
>   %6.2f — для вещественных чисел с точностью до 2 знаков, и шириной поля 6 символов (fmt.Printf("%6.2f", 3.14159)  // "  3.14")

5. Указатели (pointers)

>   %p — для вывода адреса переменной в виде указателя (в шестнадцатеричном виде) 
x := 42
fmt.Printf("%p", &x)  // 0xc0000b2008)

6. Логические значения

>   %t — для вывода булевого значения true или false
fmt.Printf("%t", true)  // true

7. Другие полезные спецификаторы

>   %v — для вывода значения в "стандартном" формате (для любых типов данных)

fmt.Printf("%v", 42)  // 42
fmt.Printf("%v", "Привет")  // Привет

>   %+v — для структур, выводит поля структуры с именами

type Person struct {
  Name string
  Age  int
}
p := Person{"Иван", 30}
fmt.Printf("%+v", p)  // {Name:Иван Age:30}

>   %#v — для более детализированного вывода (например, для структур, показывает как эта структура была бы записана в исходном коде)

fmt.Printf("%#v", p)  // main.Person{Name:"Иван", Age:30}

Пример использования:

package main

import "fmt"

func main() {
    name := "Иван"
    age := 30
    height := 182.5

    fmt.Printf("Имя: %s | Возраст: %d | Рост: %.1f см\n", name, age, height)
}
